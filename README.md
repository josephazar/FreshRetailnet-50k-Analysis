# FreshRetailNet-50K: Demand Forecasting & Inventory Optimization

End-to-end pipeline that forecasts demand for perishable retail products and optimizes order quantities using Newsvendor theory. Built on the **FreshRetailNet-50K** dataset (50,000 store-product combinations, 90 days of hourly sales data).

## Key Results

| Metric | Our Pipeline | Published Baseline (Wang et al., 2025) |
|--------|-------------|----------------------------------------|
| **WAPE** | **23.26%** | 29.02% |
| **WPE** | −1.55% | +2.58% |
| **Profit improvement** | **+203%** over static rules | — |

Our LightGBM ensemble with Tobit/IMR censoring correction outperforms the published TFT + TimesNet baseline by **5.8 percentage points** (20% relative WAPE reduction) while being simpler, faster, and more interpretable.

## Key Concepts

- **Censored demand recovery** — sales ≠ demand when stockouts occur (~44% of observations); three-stage Tobit/IMR correction recovers latent demand
- **120+ engineered features** — rolling stats, EWMA, day-of-week profiles, hierarchy clustering, lag features, cross-features
- **LightGBM ensemble** — MAE + Huber + quantile models with 4-fold expanding-window temporal CV
- **Newsvendor optimization** — 18 inventory policies (Normal, Gamma, Empirical, KDE) evaluated on a profit–service Pareto frontier
- **ABC-XYZ segmentation** — prioritizes modeling effort on high-value, predictable items

## Scripts

### Main Pipeline
```bash
python demand_forecast_and_inventory_optimization.py
```
Full end-to-end pipeline: data loading → Tobit/IMR recovery → 120+ features → LightGBM ensemble (4-fold temporal CV) → forecast evaluation → 18 inventory policies → ABC-XYZ segmentation. Runs on all 50K store-products (~84 min CPU).

### Benchmark Comparison
```bash
python benchmark_comparison.py
```
Compares **6 forecasting models** on a 3,000-SP stratified subsample:
- **Statistical**: Naive Seasonal, SSA (FreshRetailNet official baseline)
- **Neural**: DLinear (AAAI 2023)
- **Gradient Boosting**: LightGBM-Minimal, XGBoost-Minimal, CatBoost-Minimal (all with 14 basic features, no censoring correction)

LightGBM-Minimal achieves the best MAE (0.395) and profit (0.340). CatBoost is statistically indistinguishable (p = 0.714); XGBoost trails slightly. All three GBDT variants outperform neural and statistical baselines.

### Recovery Comparison
```bash
python recovery_comparison.py
```
Compares **5 demand recovery methods** and their downstream forecasting impact:
- No Recovery, Simple Mean, **Tobit/IMR** (ours), SAITS, TimesNet

Tobit/IMR achieves the best forecast bias (WPE = +1.94%) while being 40–1,100× faster than DL alternatives, with no WAPE degradation.

### Statistical Significance Tests
```bash
python statistical_significance.py
```
Bootstrap 95% CIs (10,000 resamples) and paired Wilcoxon signed-rank tests with Holm–Bonferroni correction across 3,000 SPs. Confirms LightGBM significantly outperforms all baselines (p < 0.001) except CatBoost (p = 0.714).
